# PRD (Product Requirements Document)
## 판타지 RPG 모바일 게임 — 프로젝트명: **에테르나 크로니클 (Aeterna Chronicle)**

---

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 장르 | 판타지 RPG (페트 육성 + 수집형) |
| 플랫폼 | iOS / Android (웹 기반 → Capacitor 래핑) |

## 2. 세계관 & 스토리

### 2.1 배경
마법이 실재하는 판타지 세계 **"육계(六界)"**. 세상은 여섯 속성(화·수·목·뇌·지·풍)의 힘으로 이루어져 있으며, 각 지역은 하나의 속성이 지배한다. 고대에 봉인된 **"공허(空虛)"** 가 다시 깨어나려 하고, 플레이어는 전설의 페트와 함께 세상을 구하는 여정을 떠난다.

### 2.2 지역 & 스토리 구성

| 레벨 구간 | 지역명 | 지배 속성 | 스토리 요약 |
|-----------|--------|-----------|-------------|
| 1 ~ 20 | 초원의 마을 & 고요한 숲 | 목(木) | 튜토리얼. 주인공이 페트와 처음 만나고, 공허의 징조를 발견함 |
| 21 ~ 40 | 화염 협곡 & 용암 지대 | 화(火) | 화속성 부족과 갈등. 첫 번째 봉인석 발견 |
| 41 ~ 60 | 심해 유적 & 폭풍 해안 | 수(水) | 고대 문명의 비밀. 공허의 사도 첫 등장 |
| 61 ~ 80 | 뇌격 평원 & 폐허 도시 | 뇌(雷) | 인간 세력 간의 배신과 동맹. 두 번째 봉인석 |
| 81 ~ 100 | 대지의 요새 & 지하 미궁 | 지(地) | 공허의 본거지 접근. 동료의 희생 |
| 101 ~ 120 | 풍천(風天) & 신들의 영역 | 풍(風) | 최종 결전. 공허 봉인 혹은 소멸 |

## 3. 속성 체계

### 3.1 6속성 상성 구조 (순환형)

```
화(火) → 풍(風) → 목(木) → 지(地) → 뇌(雷) → 수(水) → 화(火)
  (강함)                                                  (약함)
```

| 속성 | 강한 상대 | 약한 상대 | 특성 |
|------|-----------|-----------|------|
| 🔥 화(火) | 풍(風) | 수(水) | 공격력 특화 |
| 💧 수(水) | 화(火) | 뇌(雷) | 힐/지원 특화 |
| 🌿 목(木) | 지(地) | 화(火) | 회복력 특화 |
| ⚡ 뇌(雷) | 수(水) | 지(地) | 속도/연타 특화 |
| 🪨 지(地) | 뇌(雷) | 목(木) | 방어/탱커 특화 |
| 💨 풍(風) | 목(木) | 화(火) | 회피/디버프 특화 |

- **상성 강함**: 데미지 +30%, 명중률 +10%
- **상성 약함**: 데미지 -20%, 피격 시 추가 피해 +10%
- **중립**: 보정 없음

## 4. 캐릭터 시스템

### 4.1 캐릭터 생성
- 이름 입력, 외형 커스터마이징 (헤어·얼굴·체형 등)
- **속성 선택**: 6속성 중 1개 선택 (이후 변경 불가)
- 속성에 따라 초기 스탯 분포 및 스킬 트리 방향 결정

### 4.2 스탯 구조

| 스탯 | 설명 |
|------|------|
| HP | 생명력 |
| MP | 마력 (스킬 사용) |
| ATK | 물리 공격력 |
| MATK | 마법 공격력 |
| DEF | 물리 방어력 |
| MDEF | 마법 방어력 |
| SPD | 속도 (선공 결정) |
| CRIT | 치명타 확률 |
| DODGE | 회피율 |

### 4.3 스킬 트리
- 속성별로 고유한 스킬 트리 구성 (3개 분기, 각 분기 5단계)
- 레벨업 시 스킬 포인트(SP) 획득 → 트리에 투자
- 분기 예시 (화속성 기준):
  - **연소 계열**: 도트 데미지, 광역 화염
  - **폭발 계열**: 고폭발력 단일기, 자폭 버프
  - **지배 계열**: 적 디버프, 화속성 강화 버프

### 4.4 칭호 시스템
- 특정 업적 달성 시 칭호 획득 (스탯 소폭 보너스 포함)
- 예: "첫 포획자" (+HP 1%), "천 마리의 사냥꾼" (+ATK 2%), "공허의 정복자" (+전체 스탯 3%)
- 최대 1개 칭호 장착 가능

## 5. 페트 시스템

### 5.1 페트 기본 구조

각 페트는 **체(HP)·공(ATK)·방(DEF)·속(SPD)** 4개 스탯에 대해 독립적인 **개인 RANK**를 보유한다.
RANK는 포획·획득 시 스탯별로 개별 추첨되며, 이후 변경 불가(Fix)이다.

#### RANK 테이블

| RANK | 출현 확률 | Lv.1 보너스율 | 성장 계수 |
|------|----------|--------------|---------|
| S    | 5%       | +20%         | 1.5     |
| A    | 20%      | +10%         | 1.2     |
| B    | 50%      | ±0%          | 1.0     |
| C    | 20%      | -10%         | 0.8     |
| D    | 5%       | -20%         | 0.6     |

- 속성: **Fix** (포획/획득 시 속성 고정, 변경 불가)
- 스탯 보너스 공식: `스탯 = 종족 기본치 × (1 + RANK 보너스율)`

### 5.2 Lv.1 초기치 생성 공식

포획 또는 획득 시 각 스탯의 초기치를 다음 공식으로 결정한다.

```
체력 초기치 = 종족 기본 HP  × (1 + RANK_HP  보너스율)
공격 초기치 = 종족 기본 ATK × (1 + RANK_ATK 보너스율)
방어 초기치 = 종족 기본 DEF × (1 + RANK_DEF 보너스율)
속도 초기치 = 종족 기본 SPD × (1 + RANK_SPD 보너스율)
```

- **종족 기본치**: 페트 종족별 고정값 (별도 페트 데이터 테이블에 정의)
- 각 스탯의 RANK는 독립적으로 추첨 (예: HP=S, ATK=C, DEF=A, SPD=B 조합 가능)
- 초기치는 소수점 이하 버림(floor) 처리

### 5.3 레벨업 성장 공식

레벨업마다 각 스탯이 다음 공식으로 증가한다.

```
레벨업 스탯 증가량 = floor(종족 기본 성장량 × RANK 성장 계수)
최종 스탯(Lv.N)  = Lv.1 초기치 + Σ 레벨업 증가량 (레벨 2 ~ N)
```

- **종족 기본 성장량**: 종족별 고정값 (예: 불꽃여우 ATK 기본 성장량 = 8)
- RANK 성장 계수는 5.1 RANK 테이블 참조
- 계산 예시 (불꽃여우 ATK, RANK A, 기본 성장량 8):
  - 레벨업 증가량 = floor(8 × 1.2) = 9
  - Lv.10 ATK = Lv.1 초기치 + (9 × 9레벨)

### 5.4 페트 획득 경로 (v1.0)
1. **야생 포획**: 필드에서 Lv.1 페트 조우 → 전투 후 포획 시도
2. **퀘스트 보상**: 특정 스토리/일일 퀘스트 완료 시 Lv.1 페트 지급

### 5.4.1 야생 포획 규칙

#### 출현 조건
- **Lv.1 페트만 포획 가능** (성장한 야생 몬스터는 포획 불가)
- 필드에서 Lv.1 페트는 **매우 낮은 확률로 출현** (희귀 조우 이벤트)
- 희귀도가 높은 페트일수록 출현 확률 추가 감소

#### 포획 확률 공식 (임시 v0.2)

```
포획 확률(%) = 희귀도 기본 확률 × HP 보정 계수 × 레벨 보정 계수

HP 보정 계수   = 1 + (1 - 잔여HP / 최대HP)      → 잔여HP 0%  시 ×2.0 / 잔여HP 100% 시 ×1.0
레벨 보정 계수 = min(1 + (캐릭터Lv - 지역기준Lv) × 0.02, 1.5)
                                                   → 캐릭터 레벨이 높을수록 최대 ×1.5
```

| 희귀도 | 기본 출현 확률 | 기본 포획 확률 |
|--------|--------------|--------------|
| 일반   | 보통          | 30%          |
| 고급   | 낮음          | 15%          |
| 희귀   | 매우 낮음     | 5%           |
| 영웅   | 극히 낮음     | 1%           |
| 전설   | 초희귀        | 0.2%         |

> 기본 확률 수치는 임시값이며 밸런스 테스트 후 조정 예정

#### 도주 메커니즘
- 포획 시도 전 매 턴 종료 시 **도주 판정** 발생
- 도주 확률: 잔여 HP가 높을수록 / 희귀도가 높을수록 증가
- 도주 성공 시 전투 종료, 해당 페트 재조우 불가 (필드 리스폰 주기 적용)

### 5.5 페트 스킬
- **v1.0**: Lv.10 도달 시 스킬 1개 습득 (종류는 페트 고유)
- **v1.5 이후**: Lv.20, 30, 40, 50 구간 추가 스킬 습득 확장
- 액티브 스킬 최대 3개, 패시브 스킬 최대 2개 장착 (슬롯은 v1.5 이후 점진 개방)
- 스킬은 페트 종류 고유 (공유 불가)

### 5.6 페트 진화 (v1.5 도입)
- 특정 레벨(예: 30, 60) 도달 + 진화 아이템 소모 시 진화
- 진화 시: 외형 변화 + 스탯 대폭 상승 + 새 스킬 개방
- 최대 2단계 진화 (기본 → 1진화 → 2진화)

### 5.7 페트 도감
- 포획/획득한 페트 자동 등록
- 도감 달성률에 따른 보상 (예: 25% 달성 → 희귀 아이템, 100% → 전설 칭호)
- 페트별 서식지·특성·스토리 열람 가능

## 6. 아이템 시스템

### 6.1 일반 아이템 티어

| 티어 | 색상 | 획득 방법 |
|------|------|-----------|
| 일반 | 회색 | 필드 드랍, 상점 구매 |
| 고급 | 초록 | 던전, 퀘스트 보상 |
| 희귀 | 파랑 | 엘리트 몬스터, 거래소 |
| 영웅 | 보라 | 필드 보스, 레이드 |
| 전설 | 주황 | 특수 던전, 시즌 보상 |

### 6.2 세트 아이템

#### 세트 티어
- 세트 아이템도 일반 아이템과 동일한 5단계 티어 적용
- 상위 티어 세트일수록 세트 효과 강력

#### 세트 구성
- 1세트 = 4~6개 아이템 (투구, 갑옷, 장갑, 신발, 무기, 보조 무기)
- **속성별 세트**: 각 속성마다 전용 세트 존재 (착용 속성 일치 시 추가 보너스 +10%)

#### 세트 효과 예시 (화속성 영웅 세트 "불꽃의 군주")
| 착용 수 | 효과 |
|---------|------|
| 2개 | ATK +15%, 화속성 데미지 +10% |
| 4개 | 공격 시 30% 확률로 화염 폭발 발동 |
| 6개 (풀세트) | 외형 변환 + 궁극기 "멸화진(滅火陣)" 개방 |

#### 세트 외형 변환
- 풀세트 착용 시 캐릭터 외형 전체 변화 (세트 전용 이펙트 포함)
- 외형 변환은 옵션 설정으로 ON/OFF 가능

## 7. 전투 시스템

### 7.1 기본 전투
- 턴제 기반 전투 (캐릭터 + 페트 1마리 동시 참전)
- SPD 수치에 따라 행동 순서 결정
- 속성 상성 자동 계산 및 표시

### 7.2 전투 데미지 공식 (임시 v0.2)

> ⚠️ 스킬배율 목록 확정 후 최종 공식으로 업데이트 예정

#### 스킬 분류 및 배율 구조

| 대분류 | 소분류 | 특성 | 배율 방향 |
|--------|--------|------|----------|
| 단일기 | 공격 | 단일 대상, 높은 배율 | 단일기 > 광역기 |
| 단일기 | 상태이상 | 단일 대상 + 상태이상 부여 (독·마비·기절 등) | 낮은 배율 + 효과 |
| 광역기 | 공격 | 다수 대상, 단일기 대비 낮은 배율 | 광역 보정 적용 |
| 광역기 | 상태이상 | 다수 대상 + 상태이상 부여 | 낮은 배율 + 효과 |
| 회복기 | 회복 | 아군 HP/MP 회복 | 배율 대신 회복량 수치 사용 |

- 레벨이 높을수록 더 높은 배율/효과의 스킬 획득 (Lv.10 < Lv.20 < Lv.30 < Lv.40 < Lv.50)
- 스킬별 구체 배율 수치는 별도 스킬 데이터 테이블에 정의 예정 (추가 업데이트)

#### 기본 데미지
```
기본 데미지 = max(ATK_공격자 × 스킬배율 - DEF_방어자 × 0.5, 1)
```

#### 속성 보정 계수
| 상성 | 계수 |
|------|------|
| 강함 | 1.3  |
| 중립 | 1.0  |
| 약함 | 0.8  |

(명중률 보정은 3.1 속성 체계 참조)

#### 치명타
```
발동 확률: 공격자 CRIT 스탯(%)
치명타 데미지 계수: × 1.5
```

#### 최종 데미지
```
최종 데미지 = floor(기본 데미지 × 속성 보정 × 치명타 계수 × 랜덤(0.9 ~ 1.1))
```

### 7.3 자동 전투
- 버튼 1개로 자동 전투 ON/OFF
- AI는 MP 효율 최적화 + 속성 상성 우선 스킬 사용
- 보스 전투에서도 자동 전투 사용 가능

### 7.4 콤보 (속성 연계)
- 캐릭터 속성 = 페트 속성 일치 시: **동속성 콤보** 발동
  - 공격력 +20%, 특수 콤보 스킬 1개 개방
- 상성 강함 조합 시: **상성 콤보** 발동
  - 크리티컬 확률 +15%

### 7.5 필드 보스
- 레벨 구간마다 1~2개 필드 보스 배치
- 주기적 리스폰 (12시간 간격)
- 공격 참여자 전원에게 기여도 보상 지급
- 보스 처치 시 영웅~전설 티어 아이템 드랍

## 8. 소셜 시스템

### 8.1 길드 시스템 (v1.5 도입)
- 최대 30명 구성
- 길드 레이드 (주 1회): 전원 참여, 공동 보상
- 길드 전용 던전: 길드 포인트로 입장
- 길드 랭킹 보상 (주간)

### 8.2 결투장 PvP (v2.0 도입)
- **실시간 1:1 대전 (WebSocket 기반)**
- 세트 아이템 + 페트 조합 전략 대결
- 시즌제 운영 (4주 1시즌), 시즌 종료 시 랭킹 보상

### 8.3 거래소 (v2.0 도입)
- 플레이어 간 아이템·재화 거래
- 유료 재화 직거래는 불가 (골드만 사용)
- 거래 수수료 5% (소각 처리 → 인플레 방지)
